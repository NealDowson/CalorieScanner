# CalorieScanner - Android приложение для определения калорийности

Приложение определяет калорийность блюда по фото с помощью Google ML Kit.

## Возможности

- 📸 Съемка фото из приложения
- 🖼️ Выбор фото из галереи
- 🧠 Автоматическое распознавание продуктов с помощью ML Kit
- 📊 Отображение калорий, белков, жиров и углеводов
- 📱 Современный Material Design 3 интерфейс

## Технологии

- **Kotlin** - основной язык программирования
- **Jetpack Compose** - декларативный UI
- **Google ML Kit** - Machine Learning для распознавания изображений
- **Material Design 3** - современный дизайн
- **Min SDK 26** (Android 8.0)

## Установка и запуск

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Подключите Android устройство или запустите эмулятор
4. Нажмите Run (▶️)

## Как это работает

1. Пользователь делает фото блюда или выбирает из галереи
2. ML Kit анализирует изображение и возвращает метки
3. Приложение сопоставляет метки с базой данных продуктов
4. Отображаются результаты: калории, БЖУ, категория
